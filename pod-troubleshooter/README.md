## Overview
- Troubleshoot pods via CPU and Memory resource usage.

Compatible with local, bare-metal or cloud provided clusters (e.g. k3s, kind, minikube, Talos Linux, EKS etc..)

## Features
- CPU Total
- CPU Throttled
- CPU Resource Requests/Limits
